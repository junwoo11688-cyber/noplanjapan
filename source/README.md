# 어디든 복권 · 일본판

일본 47개 도도부현의 여행지 가운데 한 곳을 조건 없이 무작위로 뽑는 React 앱입니다. 15장의 복권 중 하나를 고르고 봉인 커버를 뜯으면 여행지가 공개됩니다.

## 주요 기능

- Web Crypto API 기반 무작위 추첨
- 복권 15장 직접 선택 및 안전한 셔플
- 마우스·터치·키보드를 지원하는 봉인 커버
- 47개 여행지별 명소, 음식, 추천 일정, 팁, 미션
- 일본 지도 위 대략적인 위치 표시
- Web Share API 및 Clipboard API 공유
- 최근 결과 최대 30개와 여행지 저장
- 미션 다시 뽑기, 효과음, 모바일 메뉴
- `prefers-reduced-motion`과 키보드 접근성 지원

## 개발

Node.js 20.19 이상을 권장합니다.

```bash
npm install
npm run dev
```

프로덕션 빌드는 `npm run build`로 `dist/`에 생성됩니다.
