どこでも旅くじ — 일본 여행 복권 사이트

[가장 간단한 실행]
1. 이 폴더의 index.html을 더블클릭합니다.
2. 별도 설치나 인터넷 연결 없이 바로 사용할 수 있습니다.

[구성]
- index.html : 단독 실행 가능한 완성본
- source/    : React + TypeScript + Vite 전체 소스

[일본판 적용 내용]
- 일본 47개 도도부현을 각각 하나의 실제 여행지로 구성
- 모든 여행지에 실제 명소 3곳, 현지 음식 3개, 추천 코스, 1박 2일 일정, 팁, 랜덤 미션 수록
- 한국어 UI와 일본 지도, JP 일련번호 적용
- 화지 질감, 쪽빛·주홍·금빛, 목판화 파도와 후지산을 활용한 에도시대 테마
- 한국판과 동일한 복권함 선택, 섞기, 봉인 커버 개봉, 결과 저장, 최근 기록, 공유, 여행 일정, 미션 다시 뽑기, 효과음 기능 유지
- 브라우저 localStorage에 최근 결과 최대 30개와 저장한 여행지를 보관

[소스 실행]
Node.js 20.19 이상이 설치된 환경에서 source 폴더를 열고 아래 명령을 실행합니다.

  npm install
  npm run dev

또는 Windows에서 source/start-site.bat을 더블클릭합니다.

[빌드 확인]
TypeScript 검사와 Vite 프로덕션 빌드를 통과했으며, 47개 여행지 데이터·랜덤 추첨·단일 HTML 패키징 검증을 완료했습니다.
